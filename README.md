## Instalación

```bash
npm install
```

## Desarrollo

### Servidor de Desarrollo
```bash
npm start
```
Navega a `http://localhost:4200/`

### Compilación para Producción
```bash
npm run build
```
Los archivos compilados se guardarán en `dist/portfolio/browser/`

## Estructura del Proyecto

```
src/app/
├── components/     # Componentes de la aplicación
├── services/       # Servicios (Theme, Data)
├── models/         # Interfaces TypeScript
└── app.component   # Componente principal
```

## Personalización

### Cambiar Datos Personales
Edita `src/app/services/data.service.ts`

### Modificar Temas de Color
Edita `src/styles.css` para ajustar las variables CSS
